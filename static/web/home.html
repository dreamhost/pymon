About pymon

pymon is an open source network and process monitoring solution implemented in python. The interface and conifiguration is designed to be easily and rapidly deployed, saving on time and overhead often associated with other monitoring solutions.

Features

  • The perfect remote monitoring tool for systems administrators and IT managers alike
  • Standard python distutils setup and installation for rapid deployment
  • Simple ini file configuration for easy maintenance
  • SQL Storage for easy access to state data by thrid-party tools
  • Asynchronous (twisted python) code for near-realtime capabilities
  • The ability to monitor thousands of simultaneous services, hosts, page content checks, responses to protocol requests, etc.
